# Jokebook 

A full-stack Node/Express web app that stores and displays jokes by category.

## Features
- View random jokes
- Search jokes by category
- Add your own joke
- View all categories

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `node server.js`
4. Open your browser at `http://localhost:3000`

## Tech Stack
- Node.js
- Express
- SQLite
- HTML/CSS/JS
